Welcome to I-HIMS!

건강 데이터 통합 관리 시스템( Intelligent Health data Integration & Management System )

📌 I-HIMS 란?

I-HIMS는 정보 제공 주체인 개인이 흩어져 있는 자신의 의료 데이터를 통합하여 관리할 수 있도록 한 웹 기반 FHIR 를 활용한 통합건강관리시스템입니다.

📌 FHIR 란?

상호운용성*을 지원하여 기존 임상정보 문서교환 표준의 단점을 보완한 차세대 문서 교환 기술 입니다.

* 상호운용성 : 하나의 시스템이 동일 또는 이기종의 다른 시스템과 아무런 제약 없이 서로 호환되어 사용할 수 있는 성질을 말합니다.
